Syntax rules: PEP-8 - Style Guide for Python Code

Print function
 |-> end attribure: By default Python prints newline after every print statement. Using this parameter, you can end a print statement with any character/string
 |-> sep attribute: The separator between the arguments to print() function in Python is space by default, which can be modified using sep parameter

Escape sequences
\newline	- backslash and newline ignored
\\		- backslash
\'		- single quote
\"		- double quote
\a		- ASCII bell
\b		- ASCII backspace
\f		- ASCII formfeed
\n		- ASCII linefeed
\r		- ASCII carriage return
\t		- ASCII horizontal tab
\v		- ASCII vertical tab
\ooo		- character with octal value ooo
\xhh		- character with hexadecimal value hh

single, double quoted strings, multi-line strings
identifiers: Giving name to variable, function, class, module, etc..
identifier rules

type function: returns type of the object
id function: returns unique id of the object

keywords: reserved words in Python language which has special meanings, and can't be used as
an identifier

Use the help() utility to display the documentation of modules, classes, functions, keywords, etc..

binary, octal, hexadecimal, complex number representation
0b010: Binary
0o123: Octal
0xf00: Hexadecimal
1+2j: complex

immutable objects: number, string, tuple, etc..
mutable objects: list, dict, set, array, etc..

type casting: convert one data type to another

input function: takes user inputs, always in string, use type casting to convert into appropriate data types
split function: split the string with specified delimiter

string formation: .format function; by default follows the order; Manipulate order by specifying {0}, {1}
